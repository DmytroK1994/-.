Повністю очистити програму?

Після підтвердження з цього пристрою буде назавжди видалено:

  • усі найменування та типові ваги;
  • поточний підрахунок;
  • всю історію підрахунків.
Цю дію неможливо скасувати. Відновити дані можна буде лише з попередньо збереженої резервної копії.

Історія підрахунків

Найменування

Масове додавання

Один рядок = одне найменування. Дублікати буде пропущено.

Пошук у найменуваннях

Швидко знайди позицію, щоб відредагувати або видалити її.

Тема оформлення

Вибрана тема збережеться на цьому телефоні.

Налаштування

Дані зберігаються в пам'яті цього браузера. Для надійності періодично роби резервну копію.

Інструкція встановлення

Перевіряю можливість встановлення на цьому пристрої…

iPhone / iOS
1. Відкрий цей додаток у Safari.
2. Натисни кнопку “Поділитися”.
3. Обери “На початковий екран”.
4. Натисни “Додати”.

Після цього “Облік залишків” з’явиться на екрані телефона як звичайний додаток.

Android
1. Відкрий опублікований застосунок через HTTPS у Google Chrome.
2. Натисни зелену кнопку “Встановити застосунок”, якщо вона з’явилася вище.
3. Підтвердь системне встановлення Chrome.
4. Якщо кнопки немає, відкрий меню Chrome ⋮ та обери “Встановити додаток” або “Додати на головний екран”.

Важливо
Локальний файл file:// не можна встановити як PWA на Android. Використовуй опубліковану HTTPS-адресу GitHub Pages.

Як оновити додаток
Якщо після оновлення щось виглядає неправильно, видали ярлик з екрана телефона і додай додаток заново.

Версія 27

Мобільний інструмент для швидкого підрахунку складських залишків. Працює без реєстрації, а всі записи зберігаються локально на цьому пристрої.

Основні можливості

  • довідник найменувань і швидкий пошук;
  • підрахунок піддонів, штук та окремої ваги;
  • історія підрахунків за датами;
  • три теми оформлення;
  • попередній перегляд і компактний PDF формату A4;
  • резервне копіювання та відновлення даних.
